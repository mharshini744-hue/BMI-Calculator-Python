# BMI Calculator using Python

## Description
This project is a simple BMI (Body Mass Index) Calculator developed using Python.

The program takes the user's weight and height as input and calculates the BMI value.

It also displays the BMI category such as:
- Underweight
- Normal weight
- Overweight
- Obese

## Technologies Used
- Python
- VS Code

## File Included
- bmi.py
- Output Screenshot

## How to Run
1. Open the project in VS Code
2. Open terminal
3. Run the following command:

```bash
python bmi.py
